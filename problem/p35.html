Windows 版本赛盾双击闪退解决教程
若您在 Windows 系统上双击赛盾 VPN 后闪退请查看本篇文章解决
近期出现在 Windows 使用 赛盾VPN 双击闪退的情况，技术部门经过调研发现是因为 Windows 更新了 KB5011048补丁导致，直接删除补丁快连即可正常使用
PS: Windows 同时更新了多个补丁，请认准 KB5011048
可以通过以下方法删除补丁，使赛盾VPN 正常使用:
方法一:
1.浏览器输入:ms-settings:windowsupdate-history 回车
2.会跳出补丁更新的历程,往下拉到最下面有个卸载更新
3.找到 KB5011048 补丁点击后即可看到卸载选项:
方法二:
1.打开控制面板→卸载程序→左侧点击查看已安装的更新;
2.找到 KB5011048 更新文件→右键点击→卸载即可:
注:若您在下载和使用过程中有任何问题，您随时可以发送邮件给 {{kfEmail}}，将由专业的客户服务人员帮助您获取到最新版本的软件。
