Host Network Service服务需要进行特殊设置
您的设备上运行的 Host Network Service，可能导致无法正常使用赛盾，这里为您提供设置 HostNetwork Service 的教程。
如果您是Dell品牌的电脑，且安装了Host Network Service软件。可能会发现无法访问网站。关闭 Host Network Service 的相关服务可以让赛盾 VPN 正常运行具体操作流程如下
1.右击最下方菜单栏，打开任务管理器
2.在任务管理器中，打开服务
3.在服务中找到“Host Network Service”,双击打开
4.启动类型改为禁用
5.服务状态更改为停止，然后点击确定
